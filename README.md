# Sellvia · ChicMentor – "Become a Fashion Creator" landing

Marketing landing for the ChicMentor / "Become a Fashion Creator" store (Sellvia, Summer Fashion Week).

- **Copy:** taken verbatim from the approved chicmentor prototype (frozen – do not edit text).
- **Design system:** based on sellvia.com/luxury-collection – SF Pro Display, dark hero with
  champagne-shimmer headline, amber accent, scroll-reveal + marquee animations.
- Fully static. Just open `index.html`.

## Structure
- `index.html` – markup
- `assets/styles.css` – all styles (incl. @font-face)
- `assets/main.js` – nav scroll-state, scroll-reveal, FAQ accordion, glow-orb
- `fonts/` – SF Pro Display (woff2, weights 400–800)
- `img/` – Sellvia logo (white for dark hero, dark for footer / scrolled nav)

Note: the "Featured on" marquee logos (Forbes / Inc. / Entrepreneur) currently load from sellvia.com.
